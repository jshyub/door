# Door selector 프로젝트

다양한 색상과 모양의 문(Door) 선택 화면 예제!

이 프로젝트는 사용자에게 문을 선택할 수 있는 인터랙티브한 웹 페이지를 제공합니다. 각 문은 호버하는 경우 색상이 변경되고 확대되는 효과를 가지고 있어 사용자에게 시각적인 흥미를 유발합니다.
